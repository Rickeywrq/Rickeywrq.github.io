How to deploy to github

blog

如何将你的hexo博客搭建到github上让更多的人看到了

创建

1.首先你要创建属于自己的github账户

github

2.创建一个项目库

点击右上角的

命名必须按照你用户名.github.io,就像下面这样

你要在Description里描述一下你的库是做什么的，例如：我的hexo博客

接下来就可以点击绿色按钮创建

设置

要在你之前创建的文件下安装一个hexo-deployer-git插件

设置

到文件的最底部

要按一下 i 进入输入模式，才能进行修改

修改完后，按esc退出

输入:wq 回车生效

部署

要输入你的账号与密码

只要等待就可以啦

然后你就可以搜索你之前的网址

https://用户名.github.io/
